# Overview
This is Hacking with Swift Day 35: Milestone (projects 4-6).

A simple app for kids to help them practice multiplication tables. Includes multiplication factors from 2 to 12, and can ask 5, 10, or 20 questions in one game.

---

List of Features:
- Uses ZStack to create a background color
- Uses UI elements such as Steppers and Pickers for changing settings
  - can change multiplication factors ranging from 2 to 12
  - can change number of questions each game asks: 5, 10, 20
- Generates random questions based on settings
